## Vue CLI 3

* Install Vue CLI 3

```sh
npm install -g @vue/cli
# use v2 init
npm install -g @vue/cli-init
```

* Prototype with Vue CLI

```sh
vue serve App.vue
```

* Generate an App

```sh
vue create my-awesome-app-v3
## compare with v2
vue init my-awesome-app-v2
```

* Use vue ui to configure an app

```sh
vue ui
```
